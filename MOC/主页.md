## 当前焦点

```TABLE 
file.mtime as "最后更新"
FROM "1-项目"
WHERE contains(file.name, "[进行中]")
SORT file.mtime DESC
LIMIT 3
```

## 最近笔记

```TABLE file.mtime as "最后修改"
WHERE file.mtime > date(today) - dur(7 days)
SORT file.mtime DESC
LIMIT 5
```
