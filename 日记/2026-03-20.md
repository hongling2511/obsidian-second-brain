## 📚 学习笔记
- Gitea 部署：binary 安装 + PostgreSQL + systemd + Cloudflare Tunnel
- Webhook 踩坑：Gitea `ALLOWED_HOST_LIST` 必须用 `loopback` 关键字，不能写 `127.0.0.1`
- Next.js 16.1.1 + Turbopack 构建部署流程
- SSH key 隔离策略：GitHub 和 Gitea 使用不同密钥对
- youtube-to-obsidian 工具流程：yt-dlp 下载 → NotebookLM 分析 → Obsidian 笔记
- 图片生成 API：聚合平台 gptsapi.net，gpt-image-1.5 可用，gemini 系列不可用

## 💡 捕获的想法
- youtube-to-obsidian 需要增加 whisper 音频转录作为 fallback，覆盖无字幕视频（106 个中 39 个因此失败）
- 批量任务需要控制并发（≤3 个子代理），避免资源争抢
- `yt2obs` 系统命令简化调用，比每次 cd + uv run 方便很多
- 图片生成应并行执行（5 张并行 ~50s vs 串行 ~5min）
